# DC-DC Buck Converter (Step-Down Converter)

## 📌 Description
This project simulates a DC-DC buck converter using MATLAB/Simulink. The converter steps down the input DC voltage to a lower output voltage using controlled switching.

## ⚙️ Features
- Step-down (buck) conversion
- PWM-based switching control
- Duty cycle variation
- Output voltage regulation

## 🛠️ Tools Used
- MATLAB
- Simulink

## ▶️ How to Run
1. Open the .slx file in MATLAB
2. Run the simulation
3. Observe input and output voltages in Scope
